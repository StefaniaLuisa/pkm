# SCAN SEO Recommendations

*Sent to SCAN early **Dec 2025** (in a recurring SEO task)*

- Brand clarity on generic queries:
  - **Title and meta:** On top pages that could rank for “save the children,” add clear brand disambiguation and value propositions like “Action Network,” “advocacy,” “policy,” “grassroots,” so searchers understand you’re the advocacy arm. Include a strong CTA-aligned meta description to lift CTR.
  - **Comparison explainer:** Publish a canonical page that explains SCAN’s role vs. Save the Children, with internal links to programs and campaigns. Use FAQ schema to answer common confusions directly in SERP.
- Own advocacy long‑tails:
  - Create or expand pillars for “advocacy,” “campaigns,” “policy,” “volunteer,” “events,” and “Washington DC.” Target queries like “save the children advocacy,” “policy action network,” “volunteer [city],” “events [month/year].” Optimize H1s and intro paragraphs with these modifiers.
- Local SEO for DC:
  - Build a DC landing hub with organization schema, NAP consistency, staff bios, and newsroom items tied to DC. Add internal links from program pages using “Washington DC” anchor variants.
- People and authority pages:
  - Strengthen the “Mark Shriver” content with a dedicated bio or author page, cross-link to relevant initiatives, and add Article/Author schema to capture those queries more reliably.
- Program content depth:
  - “Head Start,” “campaign,” “events” deserve evergreen pages with clear definitions, outcomes, and how to get involved. Add FAQ schema and, for events, Event schema with dates and locations to earn rich results.

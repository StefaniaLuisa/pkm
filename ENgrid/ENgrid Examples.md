## TNC - ENgrid Reference Pages Documentation

### **ENgrid Ecard**

Link: [~https://preserve.nature.org/page/146188/action/1?mode=DEMO~](https://preserve.nature.org/page/146188/action/1?mode=DEMO)

A simple template for eCards that uses EN’s built in functionality.

All eCards using this template can be either standalone pages, or used as embedded ecards. When setting up a page to be used as an embedded ecard you need to follow these instructions: [~https://engrid.4sitestudios.com/docs/v2/embedded-ecard~](https://engrid.4sitestudios.com/docs/v2/embedded-ecard)

### **ENgrid Advocacy Petition**

Link: [~https://preserve.nature.org/page/156636/petition/1?mode=DEMO~](https://preserve.nature.org/page/156636/petition/1?mode=DEMO)

A 2 column layout page used for Advocacy Petitions. It uses the template “4Site Page Template - Center Center 2 Column”, which contains the relevant styling for this type of page.

Other than the styling, the page features are simple. It contains:

- “FreshAddress / AtData” integration for email address validation.

- Progress bars with automatically incrementing goals.

### **ENgrid - Email To Target (Single Page, Automatically Selected Target)**

Link: [~https://preserve.nature.org/page/156852/action/1?mode=DEMO~](https://preserve.nature.org/page/156852/action/1?mode=DEMO)

An email to target action page using the page template “4Site Page Template - Center Center 2 Column” like the advocacy petition page.

This page is for a single, automatically selected target where the whole action happens on a single page. A user can land on the page, fill out their details and the Email will be sent to the targets based on the details (zip code) they provide.

If you ever wanted to offer an experience where the user chose from a list of targets, that would require a new page type to be created.

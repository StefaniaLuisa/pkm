
GA4 Parameter definitions and basics: #GTM

The following parameters are collected by default with every event, including custom events. Th automatically collected events that don't have parameters listed in the table below only receive following parameters: 
language page_location page_referrer • page_title •• screen_resolution

- page_location = The full URLe visitor was on when the event took place
- [page_referrer](https://www.rootandbranchgroup.com/previous-page-google-analytics/) = The URL of the prior page that the visitor was on before the current page
- page_title = The title tag of a page which comes from the &lt;title&gt; HTML element. For example, the page_title of this page is GA4 Event Parameters: A Pracl Guide

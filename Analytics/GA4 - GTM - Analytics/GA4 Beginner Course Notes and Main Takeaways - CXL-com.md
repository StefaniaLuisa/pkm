Report: 
**User Acquisition** shows the first session data of a user - the first interaction
**Traffic Acquisition** is the more commonly recommended report where you can see where your engaged audience is interacting with your website. 

An active user can have several definitions.
1. First, if I arrive and I have that first visit event,i.e. 
* I'm a new user, then by definitionI am an active user for that session.
2. Or a user can have an engaged session
* So again, the engaged session is 10 or more seconds, a conversion event or two or more page views
If any of those things happen while this user is on your website, then they are an active user or/and
      3. There's an engagement time parameter that gets added to a subsequent event.
**An active user** is any user who has an engaged session for the first visit eventor the engagement time millisecond parameter on a subsequent event like this scroll event.

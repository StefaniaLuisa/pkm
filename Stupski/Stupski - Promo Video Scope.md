
# DETAILED SCOPE OF WORK: PROMOTIONAL VIDEO & SLIDESHOW ELEMENTS

## PROJECT OVERVIEW

Development of two promotional elements for Stupski Foundation's website to support Glenn's book "Control" launch: (1) a video promo box and (2) an animated slideshow. Both elements will maintain brand consistency while creating engaging, minimalist visual experiences that emphasize the book title and author.

## DELIVERABLES

### 1. VIDEO PROMO BOX

#### Design Specifications

- Minimalist design focusing on the video content
- Clean layout with ample white space
- Bold typography for limited text elements
- Color scheme matching Stupski Foundation brand guidelines
- Subtle background elements that don't distract from video

#### Functional Requirements

- Auto-play functionality when element appears in viewport
- Initially muted audio with user control to enable sound
- "Learn more" button with hover state effects
- Responsive design adapting to all screen sizes
- Custom mobile version with portrait orientation optimization
- Video swap capability between desktop/mobile versions

#### Technical Implementation

- Custom CSS for styling consistent with site aesthetic
- JavaScript for video behavior control
- Performance optimization for video loading
- Integration with existing WordPress theme
- Browser testing across Chrome, Safari, Firefox, and Edge

### 2. ANIMATED SLIDESHOW

#### Design Approach

- Video mood board creation showing 3-5 animation style options
- Adaptation of selected MotionRate template to Stupski branding
- Focus on large, impactful typography for "Control" book title
- Clean transitions between scenes (fade, slide, scale)
- Integration of Stupski brand elements including dot patterns
- Color palette coordination with upcoming book cover

#### Animation Sequence

1. Opening scene featuring large "Control" text with subtle animation
2. Transition to book cover reveal with elegant movement
3. Introduction of Glenn as author with side-by-side layout
4. Final call-to-action with subtle animation

#### Technical Specifications

- After Effects animation development
- Export optimization for web performance (compressed file size)
- HTML5 video format with fallback options
- Mobile-responsive design with orientation-specific versions
- Loading optimization to prevent page performance issues

**Styling and Specifications:**

- **Video Promo Box:**

  - Clean, minimalist design with focus on large text

  - Auto-play functionality without audio initially (like Break Fake Rules hero)

  - Simple layout with embedded video and "learn more" button

  - Limited text to keep focus on bold words/phrases

  - Mobile version needed (different from desktop version)

  - Ability to swap videos between mobile and desktop versions

- **Slideshow:**

  - Using MotionRate templates as foundation

  - Focus on minimalist text design with large, bold words (especially "Control" as the book title)

  - Tayo will create a video mood board showing animation approach

  - Preference for clean transitions and not overly flashy effects

  - Will feature the book cover when available

  - Potential for side-by-side layout with Glenn on one side as the author

  - Sequential scenes: starting with large "Control" text, animating to book cover, then introducing the author

  - Stupski brand elements like dots could be incorporated

  - Avoiding text-heavy design in favor of impactful visuals

Both elements will be designed to match the upcoming book cover design (expected in September) and maintain brand consistency while being eye-catching without being overwhelming.

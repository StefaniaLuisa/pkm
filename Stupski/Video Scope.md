
# PROJECT DESCRIPTION: "CONTROL" BOOK PROMOTIONAL ELEMENTS

Development of dynamic promotional elements for Glenn's book "Control" on the Stupski Foundation website. This project includes two main components: (1) a video modal lightbox featuring a teaser video and (2) an animated slideshow. Both elements will maintain the Stupski Foundation's brand identity while creating engaging, minimalist visual experiences that emphasize the book's critique of philanthropy.

## COMPONENT 1: VIDEO MODAL LIGHTBOX

### Design Specifications

- Clean, minimalist design with focus on the embedded video
- Full-width/height video container within the lightbox
- Close button in the upper right corner
- Free-floating "Learn More" button below the video
- Custom colors and button label to match Stupski branding
- Overlay that dims the background content when active

### Functional Requirements

- 15-second teaser video plays automatically when opened
- Initially silent with user option to enable audio
- Responsive design with separate mobile-optimized version
- Video swap capability between desktop/mobile versions
- Smooth opening/closing animations

### Technical Implementation

- Custom CSS for styling consistent with site aesthetic
- JavaScript for modal behavior and video control
- Cross-browser compatibility (Chrome, Safari, Firefox, Edge)
- Performance optimization for quick loading
- Estimated development time: 3-4 hours

## COMPONENT 2: PUSHDOWN BANNER/STICKY MINI-BAR

### Design Specifications

- Initial pushdown banner at top of page
- Transforms into a compact sticky mini-bar on scroll
- Consistent branding with the modal lightbox
- Minimalist design focusing on the book title "Control"
- Custom colors matching Stupski Foundation palette

### Functional Requirements

- Smooth transition from banner to mini-bar during scroll
- Sticky positioning that remains visible as user navigates
- Call-to-action button to open the modal lightbox
- Responsive design adapting to all screen sizes
- Custom mobile version with appropriate sizing

### Technical Implementation

- Custom CSS for styling and transition effects
- JavaScript for scroll behavior and state changes
- Testing across devices and browsers
- Performance optimization to prevent page lag
- Estimated development time: 4-6 hours

## COMPONENT 3: ANIMATED SLIDESHOW

### Design Approach

- Based on "Dynamic Corporate" and "Minimalist Text Design" templates from MotionRate
- Focus on large, impactful typography for "Control" book title
- Clean, professional transitions between scenes
- Integration of Stupski brand elements including dot patterns
- Color palette coordination with upcoming book cover design

### Animation Sequence

1. Opening scene featuring large "Control" text with subtle animation
2. Transition to book cover reveal with elegant movement
3. Introduction of Glenn as author with side-by-side layout
4. Final call-to-action with subtle animation

### Technical Specifications

- After Effects animation development
- Export optimization for web performance
- Mobile-responsive design with orientation-specific versions
- Silent animation with option to add subtle background music if desired
- Similar in concept to the RAN campaign promotion example

## DESIGN PHILOSOPHY

- Emphasize simplicity to align with the book's critique of philanthropy
- Focus on bold typography with "Control" as the central visual element
- Minimal text to maintain visual impact
- Clean transitions rather than flashy effects
- Consistent branding across all promotional elements
- Eye-catching without being overwhelming

## DELIVERABLES

1. Video Modal Lightbox with embedded 15-second teaser video
2. Pushdown banner/sticky mini-bar implementation
3. Animated slideshow for embedding on the book page
4. Mobile-optimized versions of all elements
5. Documentation for content updates and management

### Personalized homepage experience:

- [x] As you’ve seen from prior recommendations, we’re recommending that Amnesty explore implementing  Lytics. This is a tool we’ve been implementing with organizations to help create a more cohesive and effective user experience. Lytics is a CDP, or Customer Data Platform, which means it consolidates data from multiple platforms into user profiles that can be used to personalize a user’s experience. The data we’re talking about are things like whether a user has donated, whether they’ve signed up, which campaigns they’ve responded to. Currently, the Amnesty USA website presents the same exact content and CTAs to everyone, whether they’ve never visited the site or are our most reliable and committed supporters. A CDP like Lytics acts as a centralized data repository that can drive personalization, even for new (anonymous) users. We’re recommending Lytics because it’s effectively free for pilot programs; and paid tiers are competitively priced. And it’s easy to set up and use.
